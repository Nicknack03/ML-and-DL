## 🚀 Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/Nicknack03/ML-and-DL.git
   cd ML-and-DL
   ```
2. Install common dependencies
   ```bash
   pip install numpy pandas scikit-learn tensorflow matplotlib seaborn jupyter
   ```
3. Launch Jupyter and open any notebook
   ```bash
   jupyter notebook
   ```
